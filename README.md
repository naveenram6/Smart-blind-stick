# Smart Blind Stick

## 📌 Overview
The Smart Blind Stick is an assistive device designed to support visually impaired individuals by detecting obstacles using ultrasonic sensors. The system provides real-time alerts, improving safety and navigation.

## 🎯 Objective
To develop a low-cost, efficient, and portable solution that enhances independent mobility for visually challenged people.

## ⚙️ Working Principle
The ultrasonic sensor emits sound waves and calculates the distance of obstacles based on the reflected signal. When an object is detected within a certain range, the system alerts the user using a buzzer.

## 🛠️ Components Used
- Ultrasonic Sensor
- Microcontroller (Raspberry Pi Pico )
- Buzzer
- Power Supply

## 🚀 Features
- Real-time obstacle detection  
- Simple and cost-effective design  
- Portable and user-friendly  
- Improves safety and navigation
